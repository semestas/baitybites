Saran Penyajian

Produk kami dikirim dalam bentuk frozen untuk menjaga kualitas. Ikuti langkah berikut untuk penyajian terbaik:

1
Defrost: Keluarkan dari freezer dan diamkan di suhu ruang atau chiller hingga tidak membeku.
2
Panaskan Minyak: Siapkan minyak goreng segar dan panaskan dengan api sedang.
3
Goreng Api Kecil: Masukkan risol dan goreng dengan api kecil agar matang merata hingga ke dalam.
4
Jangan Terlalu Lama: Goreng hingga kuning keemasan, jangan terlalu lama agar tidak pecah.
5
Sajikan Hangat: Tiriskan sejenak dan sajikan selagi hangat untuk kerenyahan maksimal.
